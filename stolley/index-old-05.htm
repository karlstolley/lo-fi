Preamble

The days of designing webpages to be mere webpages have ended. So have the days of using WYSIWYG editors, which clutter the structure of a document with a web writer’s visual choices.

Web production now stands on the threshold of a new era, the era of the “semantic Web” (which will soon be so synonymous with digital discourse as to be a redundant term).

This is an era of XML and structural documents, an era of increasingly diverse and ubiquitous web access that goes far beyond the familiar portal of the desktop browser (which is what one purportedly “sees” in a WYSIWYG editor).

This is an era when devices will need to be able to interpret and manipulate web-available discourse as much as, if not more than, humans.

This is an era when the lo-fi technology of plain text and the text editor, combined with fluent literacy in the languages supporting digital discourse, must supplant hi-fi technologies like the WYSIWYG editor and software that produces all-in-one documents like the PowerPoint presentation, the PDF, and the Flash movie.

Definining L.O.-F.I.

  • Lossless: Discourse presented through lo-fi production technologies neither degrades nor becomes trapped in the production itself. Text migrates and transforms from a single source; images, video, and other media maintain their integrity as individual files.
  • Open: Lo-fi production technologies make their source code and materials available for inspection, revision, and extension outside of the scope of any one piece of production software. This includes, by default, end-user/reader customization and repurposing.
  • Flexible: Content presented using lo-fi production technologies can be experienced unobstrusively in different ways by different users.
  • In(ter)dependent: Lo-fi production technologies direct orchestration (like a recipe), not composition (like a TV dinner), allowing users’ devices to interpret the production, sort of the way a chef interprets a recipe.

Manifesto

01 Begin solving communication problems with the lowest lo-fi technology.

Chances are that there is a plain-text solution to most production problems, whether structural, design, or interactive. The lower-fi solution is always preferable, because it will always be the most flexible to edit, revise, and extend later on.

02 The investment to learn a language has a higher return than any software.

Computer language skills, including the technical vocabulary one learns from them, are inherently extensible to other languages while fostering dialogue among producers in a way that point-and-click production software cannot.

03 All digital discourse should be editable in a simple text editor.

Embrace the simplicity and austerity of the text editor. No buttons, just content. And code.

04 Code is just as important as content.

Sorry, there’s just no getting around this. Code matters. It makes digital communication possible, and will be absolutely essential to successful digital discourse in the semantic web.

05 Without meaningful structure, digital discourse is useless.

Any digital writing without at least a touch of XML is already obsolete.

06 Accessibility is neither an afterthought nor a clean-up job.

Lo-fi methods that emphasize structured content, are always already accessible.

07 There may be no greater production transgression than File > Import.

To import a file of one media type into a file of another media type destroys the imported file’s integrity while also guaranteeing that users will need some sort of plugin, specialized software, or particular device to view the content. Harness the end user’s computing power to blend and render mixed media...and save yourself from time-sucking edits later on.

08 Incompatibility always results from a lack of foresight.

Before hitting the save button on anything, research the file type or language you are using.

09 Do not require anything of any user.

Require no plugins, no browsers or devices. Period. But...do not be hesitate to provide richer experiences to users who do have special plugins, browsers, or devices.

10 Every six months, rethink everything.
